This project is a small Todo application converted to a Vite + React setup.

Features:
- Add todos
- Toggle complete
- Edit inline (double-click)
- Delete todos
- Clear completed
- Filter (All / Active / Completed)
- Persistence using localStorage
+
Getting started (PowerShell)
+
```powershell
# from the project folder
npm install
npm run dev
# open the address printed by Vite (typically http://localhost:5173)
```

Build & preview

```powershell
npm run build
npm run preview
```

Notes

- The React entry is `src/main.jsx` and the main app component is `src/App.jsx`.
- Todos are stored under localStorage key `simple_todos_v1`.
